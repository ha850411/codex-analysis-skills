# 世足輸出模板

完整分析使用此模板。若使用者要求簡版，可保留核心欄位並壓縮文字。

## 目錄

- 賽程與單場資訊
- 出線情境、名單、數據與戰術
- 模型、建議、風險與總結
- 賽後檢討

所有機率與信心度數字都必須以 `%` 結尾，例如 `42%`，不得只寫 `42`。
