# 世足推薦閘門

這份規則用在模型完成後、輸出下注建議前。目的不是提高文字漂亮度，而是避免把世界盃小樣本、低信心、負期望值或高度相關的選項包裝成推薦。

## 1. 期望值門檻

先把市場賠率轉成損益兩平機率：

市場賠率必須先以十進位表示。來源若是美式、分數、香港盤、馬來盤、印尼盤或其他格式，只能內部轉換，不得在報告中輸出原格式。

```text
損益兩平機率 = 1 / 十進位賠率
模型優勢 = 模型機率 - 損益兩平機率
期望值 = 十進位賠率 * 模型機率 - 1
```

推薦規則：

| 市場狀態 | 輸出規則 |
| --- | --- |
| `模型優勢 < 0` 或 `期望值 < 0` | 禁止寫「價值」「可小注」「主推」；只能寫避開 |
| `模型優勢 0 到 3 個百分點` | 無下注；可寫傾向觀望，但不得給注碼 |
| `模型優勢 3 到 5 個百分點` | 只允許 0.25 個投注單位內條件單，且信心度不得高於 60% |
| `模型優勢 5 到 8 個百分點` | 可小注 0.25 到 0.5 個投注單位，但需通過先發、賽制與風險檢查 |
| `模型優勢 > 8 個百分點` | 才能考慮主推；仍需通過信心度、相關市場與比分情境檢查 |

若沒有即時賠率，只能輸出「價格門檻」，不能直接給主推。例如：`小於 2.5 球需 1.72+ 才有價值`。

## 2. 世界盃信心度閘門

| 信心度 | 允許輸出 |
| --- | --- |
| `< 55%` | 觀望 / 不下注 |
| `55% 到 59%` | 傾向觀望或 0.25 個投注單位內條件單，不得寫主推 |
| `60% 到 64%` | 小注，除非期望值大於 8 個百分點否則不得超過 0.5 個投注單位 |
| `65% 到 69%` | 可正常推薦 0.5 個投注單位，但需資料完整 |
| `70%+` | 只有正式先發、關鍵傷停、出線/晉級情境、盤口與模型一致時才可出現 |

信心度是資料品質與模型一致性的評分，不是勝率。若正式先發未確認、盤口過期、主要來源互相衝突、出線情境未算清、或市場結算範圍不明，信心度上限 60%。

## 3. 階段別降溫

- 小組首戰：熱門勝率、讓球信心與大勝比分要下修；實力接近時提高和局底線。
- 小組第二輪：首戰內容要納入，但不可只因比分過度反應。若弱隊防守可重複，熱門讓球信心上限 60%。
- 小組第三輪：先算出線路徑。若一分、少輸或淨勝球有價值，不可自動假設落後方會冒進。
- 淘汰賽：分清 90 分鐘、延長賽、PK 與晉級市場結算範圍。若市場只結算 90 分鐘，延長賽或 PK 優勢不能直接變成三路盤勝率。
- 主辦國或強烈球迷優勢：可以加權，但也要檢查媒體壓力、開幕戰情緒與裁判/VAR 放大波動。

## 4. 相關市場降重

高度相關市場不得堆疊成多個推薦：

- `小於 2.5 球` 與 `雙方進球：否`
- 熱門 `-1` 與熱門 `贏球且不失球`
- `大於 2.5 球` 與 `雙方進球：是`
- 和局與小分
- 淘汰賽 90 分鐘和局與 `進入延長賽`

若兩項都看好，必須比較期望值，只保留模型優勢較高的一項。另一項只能放在「同方向觀察」，不得給獨立注碼。

## 5. 低比分偏誤檢查

世界盃常見低比分，但不可把「大賽保守」直接等同小分。主推小分前必須檢查：

- 早段進球後，落後方是否會被迫打開？
- 雙方是否都有定位球、反擊、速度型替補或邊路傳中路徑？
- 是否有主辦國主場情緒、開幕戰節奏、媒體壓力或球迷推動？
- 中衛速度、門將出球、單後腰保護是否存在明顯失誤風險？
- 裁判尺度、VAR、點球與紅牌風險是否高於一般比賽？
- 第三輪或淘汰賽是否因出線/晉級情境使某一方必須冒險？

若至少兩項成立，小分信心度上限 59%，除非預期進球均值、正式先發與市場價格都明顯支持。

## 6. 比分情境壓力測試

每場至少列三個狀態：

| 狀態 | 要檢查的事 |
| --- | --- |
| 0-0 到 30 分鐘 | 是否真的慢？還是只是雙方不敢冒進但定位球很多？ |
| 熱門方先進球 | 對手打開後，總進球與雙方進球機率是否快速上升？ |
| 下狗先進球 | 熱門方壓上後，是否形成反擊互打或角球/點球風險？ |

若任一狀態能合理導向 3 球以上，不能把 `小於 2.5 球` 列為強推薦。

## 7. 輸出清晰度

- 每場最多一個主推。
- 若信心度低於 60%，該場預設為「無主推」。
- 精確比分要用分布語言，例如 `最高機率 1-1；替代 1-0、2-1`，不要讓第一個比分看起來像唯一預測。
- 總結表要分開 `模型傾向` 與 `可下注推薦`。模型傾向不等於可下注。
- 總結表必須保留 `雙方進球` 與 `各隊至少 1 球` 欄位；多場輸出、簡版輸出或表格太寬時也不得刪除。`各隊至少 1 球` 要分別列官方主隊與官方客隊的個別機率，不可用 `BTTS` 取代。
- 若推薦與三路盤方向不同，必須明確寫：`勝負不下注，僅看總進球/雙方進球`。
- 淘汰賽報告必須明確寫清楚推薦市場的結算範圍，例如 `90 分鐘`、`含延長賽`、`晉級` 或 `PK`。
