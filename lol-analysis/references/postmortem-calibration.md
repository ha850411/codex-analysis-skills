# 賽後檢討與模型校準

使用者要求檢討錯誤預測、回測賽前判斷、或比較預測與實際結果時，使用這份流程。

## 1. 先重建事實

- 核對比賽：賽區、階段、日期、賽制、版本、先發。
- 核對結果：系列比分、每局勝方、選邊、每局時間、主要 BP。
- 資料來源優先順序：LoL Fandom/Leaguepedia、官方 VOD、官方賽後資料、可信數據站。
- 如果只能取得部分 VOD 或摘要，需標記限制。

## 2. 重建原預測

- 原推薦勝方與精確比分。
- 原始勝率、精確比分分布、信心度、建議注碼。
- 原本支撐預測的核心假設：近期狀態、版本適性、BP 優勢、對位優勢、BO5 調整、選邊、心理/賽程。

## 3. 判定失準等級

| 等級 | 定義 | 處理 |
|---|---|---|
| 小偏差 | 勝方正確，比分差 1 局 | 檢查精確比分分布是否過窄 |
| 中偏差 | 勝方錯誤，但系列接近，例如 2-3 / 1-2 | 檢查勝率是否過度集中 |
| 大偏差 | 勝方錯誤，且比分相差至少 2 局，例如預測 A 3-2、實際 B 3-1 | 啟動信心度降溫與權重校準 |
| 地圖分布大偏差 | 勝方正確，但主推打滿/敗方 +1.5/敗方至少一局，實際被橫掃 | 啟動地圖市場降溫與取圖路徑重審 |
| 方向錯但地圖市場命中 | 勝方錯誤，但弱方 +1.5、Over 或雙方拿圖命中 | 修正勝率分布與主結論，不推翻取圖模型 |
| 系統性偏差 | 同類隊伍或同賽區連續錯誤 | 重新檢查資料來源、近期權重與模型假設 |

## 3.1 地圖分布特判

即使勝方方向正確，以下情況仍視為嚴重校準問題：

- 原預測主推打滿/地圖大分/敗方 +1.5，實際為橫掃。
- 原預測雙方至少贏一局高於 80%，實際其中一方 0 局。
- 原預測 3-0 或 0-3 機率低於 10%，但賽前已有兩項以上橫掃風險因子。

處理方式：

- 分開檢討勝方機率與地圖分布，不要用「勝方有看對」掩蓋地圖市場錯誤。
- 回頭列出敗方取圖路徑是否只是 H2H、名氣、排名或 BO5 直覺。
- 下一次同類 matchup，先套用 `recommendation-gates.md`，再決定是否能推打滿或 +1.5。
- 若原模型給敗方至少一局很高機率但實際被橫掃，檢查取圖路徑、局間相依性與橫掃情境是否漏算；只有確認可重複的流程錯誤時才調整後續模型，不因一次結果直接設 70–75% 上限。

## 4. 錯誤歸因框架

逐項判斷，不要只用「爆冷」帶過。

- 資料錯漏：名單、版本、賽制、選邊、賽程或狀態資訊是否過期。
- 近期權重錯配：是否過度相信整季戰績、過往交手或隊伍名氣，而低估最近 1–2 週內容。
- BP/版本誤判：是否低估某隊的版本答案、英雄池封鎖能力、紅藍方差異或針對 ban。
- 對位誤判：哪一路或哪組輔野/中野對位實際成為破口。
- BO5 調整誤判：是否低估勝方局間修正、連續施壓同一弱點、或敗方無法解題。
- 執行與收尾誤判：前期設計、物件交換、視野紀律、團戰站位、巴龍/龍魂處理是否與賽前假設相反。
- 市場/品牌偏誤：是否因 T1、Gen.G、BLG、G2 等高人氣或強歷史隊伍而高估穩定性。
- 合理變異：若主要來自單局重擊、偷巴龍或極端失誤，標記為變異；但不能用變異掩蓋多局重複問題。

## 5. 大偏差後的校準規則

- 將該場標記為校準警報，計算當時分配給實際結果的 Brier／log loss 貢獻；只有找到資料、版本、BP、權重或分布過窄的流程錯誤時，才針對相關信心組成下修。
- 推薦注碼降一級，例如 1–2u 改為 0.5–1u，0.5–1u 改為 0.25–0.5u。
- 重建精確比分主分布；若合理情境確實漏掉，再提高對手 3-1／3-2 或 2-1，不為了賽後結果任意加寬。
- 未完整檢查 VOD/BP 前，不得給「高信心」。
- 下一次分析相同隊伍時，必須先列出上一場大偏差的修正重點。
- 若上次錯在「打滿/至少一局」但實際橫掃，下一次相關市場必須先找到兩條以上敗方取圖路徑；找不到就不下注，即使獨贏方向仍可小注。

## 5.1 地圖市場大偏差後的校準規則

- 勝方方向看對但地圖市場錯時，獨贏模型不必全盤推翻；地圖分布模型必須降溫。
- 下一次同類 BO5，敗方 +1.5、地圖大分與雙方各取一局先降一級注碼，直到取圖路徑完成重建；模型信心度只依五項組成重算。
- 不論市場價格，先把熱門方 3-0 作為主動情境檢查，而不是機械尾端；其機率由重建後的逐局情境推導，不硬塞入 15–25% 區間。
- 原預測若把熱門疲勞、敗者組連戰或剛打滿五局解讀為劣勢，賽後需檢查這些因素是否其實轉化為手感、版本資訊與 BP 隱藏牌優勢。
- 推薦敗方 +1.5 前，必須列出「敗方取圖路徑」與「熱門方橫掃路徑」各至少兩條；若只能列出一邊，不下注。

## 5.2 隔日連戰與 3-2 勝利校準

- 回看前一天 BO5 時，先標記系列賽形狀：一路領先、被追平、逆轉、或 1-2 落後後連拿兩局。
- 若勝方 G4/G5 明顯換出更穩 BP 或不同 carry 點，這不是單純疲勞訊號，而是局間解題成功。
- 若決勝局擊殺、經濟或物件呈單方壓制，隔日不可只因比分 3-2 而把勝方評為「狀態不穩」。
- 下一場分析要把「剛展示的解題方案」拆成可延續與不可延續兩類；Hard Fearless 下尤其要檢查是否仍有足夠英雄池支撐。

## 5.3 方向錯但地圖市場命中校準

- 若主勝方錯誤，但 Over、弱方 +1.5 或雙方至少一局命中，不要把整個分析判為無效；取圖路徑可能是對的。
- 檢查主結論是否與投注建議互相打架：例如主結論寫熱門 60%+，但最佳投注其實是弱方 +1.5。
- 下一次同類 BO5，若弱方 +1.5 是主推，檢查同一精確比分分布是否合理反映弱方連拿兩局與拿下 G5 的條件；不得固定提高弱方 ML／3-2 5–8 個百分點。
- 信心度修正聚焦於實際暴露的資料或模型穩定性問題，不因某個市場命中就保留、也不因獨贏失準就全面扣分。

## 6. 檢討輸出格式

1. 賽果確認與資料來源。
2. 原預測 vs 實際結果。
3. 被推翻的賽前假設。
4. 主要錯誤歸因。
5. 模型修正清單。
6. 下一次遇到同隊伍/同類型 matchup 的操作規則。
