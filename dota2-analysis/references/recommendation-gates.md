# Dota 2 推薦閘門與局數分布校準

推薦獨贏、+1.5 maps、總局數、BO2 平局、不敗、雙方各取一局或精確比分前，使用這份檢查。目標是避免把「勝方小優」誤寫成「一定掉一局」，或把 Dota 2 的高變異當成下注理由。

## 0. 市場分歧閘門

當模型勝率與市場熱門 implied probability 相差 10 個百分點以上，或熱門方 ML 約 1.45 以下時，先回答：

- 市場是否可能已反映更完整的 stand-in、patch、draft pool、賽程或 scrim 傳聞？
- 模型是否把「獨贏有價格」誤轉成「敗方一定能拿局」？
- 敗方 +1.5 或至少一局是否有兩條可驗證取局路徑，而不是只依賴隊名、舊 H2H、明星選手或「Dota 很容易爆冷」？
- 熱門方 2-0/3-0 的路徑是否完整列出，且有具體理由說明為何不會發生？

若答案不完整：

- 獨贏價值與局數市場分開評估。
- +1.5、Over maps、BO2 平局預設不推薦或只看 live / post-draft。
- 熱門方橫掃機率不得機械性壓低。

## 1. 取局路徑閘門

推薦 BO3/BO5 大分、+1.5、雙方各取一局、BO2 1-1 前，雙方都要有至少兩個獨立取局路徑：

- Draft 路徑：能拿到版本 OP、signature hero、flex pick、last pick counter 或可懲罰對手英雄池的組合。
- 分路路徑：safe/off/mid 至少一處可穩定建立線權、rune 控制或經濟起手。
- Timing 路徑：第一波 BKB/Blink/Orchid/Pipe/Mek/Aegis 等 timing 能主動打出節奏。
- Roshan 路徑：能控視野、打盾、用 Aegis 上高，或阻止對手第一盾滾雪球。
- Late-game 路徑：高地防守、買活、團戰反手、split-push 或超後期 carry matchup 有明確條件。

以下理由不算通過：

- 「隊伍底蘊」「明星選手會醒」「BO3 很常 2-1」「Dota 變數大」。
- 只靠對手送 Roshan、50/50 搶盾、單次高地失誤或極端 throw。
- 舊版本 H2H，但沒有同 patch BP/VOD 支撐。

## 2. Draft 後重算閘門

若 draft 已公布或使用者提供 Game 1/2 draft，不得沿用賽前系列賽分布。必須重算：

- 三路 matchup 與哪一路最可能崩線。
- 雙方 timing：誰在 10/15/20/30 分鐘較強。
- 團戰先手與反手是否足夠。
- Roshan 速度與 Roshan pit 控制。
- 高地與買活條件。
- 是否有 cheese pick、無解 last pick 或陣容缺傷害/缺控/缺塔傷。

若 draft 明顯推翻賽前假設，需直接調整單局勝率與系列賽比分分布。

## 3. BO2 特別閘門

BO2 常見於小組賽，不能用一般 BO3 心態處理：

- 必須輸出 2-0 / 1-1 / 0-2 機率。
- 若推薦 1-1 或受讓，需說明雙方各自在哪一種 draft/選邊下拿局。
- 若一方英雄池、分路與節奏都明顯優勢，不能因 BO2「容易互拿」就硬推平局。
- BO2 第一局結果會影響第二局 draft adaptation；若賽程允許 live，優先建議等 Game 1 draft 或結果後再進場。

## 4. 橫掃風險閘門

出現以下兩項以上時，必須提高 2-0/3-0 或 0-2/0-3 機率，避免高信心推薦大分或敗方 +1.5：

- 敗方近期同 patch BP 被連續針對，核心英雄池被封鎖後沒有替代方案。
- 中路或雙邊分路長期落後，導致第一個 item timing 與 Roshan 視野失控。
- 對手近期同賽事已展示兩套以上穩定 draft 解法。
- 敗方需要高難度 combo 或低容錯團戰才能贏，近期執行不穩。
- 領先時高地、買活、Aegis 使用反覆出錯。
- stand-in、角色對調或 draft caller 變動使溝通與 BP 深度不穩。

## 5. 讓分與爆冷一致性閘門

當推薦弱方 +1.5、Over maps、BO2 平局或雙方各拿一局時，先回答：

- 弱方的取局路徑是否也可能連續發生兩次，形成 upset？
- 若弱方能拿一局，為什麼不能拿下系列賽？限制是 hero pool、late game、Roshan、高地還是 shot-calling？
- 熱門方若輸掉一局，下一個勝局的可靠來源是什麼？
- 精確比分分布是否把弱方 2-1 / 3-2 放進可見區間？

若最佳投注其實是弱方受讓，主結論不要寫成熱門穩勝；應改成「熱門小優但局數接近」或「弱方具備 live upset path」。

## 6. 注碼降溫條件

符合任一項時，下注建議降一級或改觀望：

- Patch 剛改，雙方同 patch 樣本不足。
- 先發、stand-in 或角色分配未確認。
- Draft 影響極大，但賽前還沒有 Game 1 draft。
- BO1 或低 tier online。
- 模型方向與可靠市場/數據站明顯相反，且缺少足夠反證。
- 推薦高度依賴單一英雄或單一 player carry，而該英雄可能被 first phase ban。
