## Valorant 賽後失準檢討模板

1. **賽果確認與資料來源：** 賽事、日期、BO、Patch、系列比分、每張地圖比分、veto、資料狀態。
2. **原預測 vs 實際結果：** 勝方、精確比分、Win%、至少一圖、信心度。
3. **被推翻的賽前假設：** Patch、同賽事樣本、H2H、post-veto 訊號、特務陣容、手槍/經濟、教練修正。
4. **地圖逐張復盤：** pick owner、選邊、關鍵回合、原本誤判點。
5. **錯誤歸因：** 依 `references/postmortem-calibration.md` 分類。
6. **模型修正清單：** 下次同隊伍 / 同 Patch / 同類 BO5 / 已公布 veto 場景的機率與信心度上限。
