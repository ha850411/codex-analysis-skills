# Valorant 搜尋檢核表

每次分析前檢查：

1. 日期是否已換算成台灣時間 UTC+8？
2. 是否已搜尋 VLR 當日賽程？
3. 是否已用 Liquipedia 或官方賽事頁交叉查核？
4. 是否已列出所有找到的當日比賽，而不是只分析一場？
5. 每場 BO1 / BO3 / BO5 是否確認？
6. 若只找到少量場次，是否已用賽事名稱再搜一次？
7. 是否確認先發 5 人、主 Duelist、Controller、IGL/Caller、stand-in？
8. 是否確認近期 Patch / Map Pool 是否影響舊數據？
9. 若核心特務或主 Duelist 受 Patch 影響，是否下修舊資料並限制信心度？
10. 是否檢查雙方同 Patch / 同賽事樣本，避免低估熱手隊伍？
11. 最近 H2H 是否跨 Patch、跨賽制或跨地圖池，是否避免過度錨定？
12. 若可靠賠率、排名或數據站與模型方向相反，是否列出反向訊號與信心度修正？
13. 是否確認當前賽事可用 Map Pool，並逐張分析兩隊每一張地圖？
14. 逐圖分析是否包含兩隊樣本、攻守表現、常用特務陣容、對位重點、單圖傾向與 Veto 意義？
15. 是否完成 Map Veto 推演，並檢查 2-0 / 反掃風險？
16. BO1 是否避免輸出 +1.5 maps？
17. 若引用賠率或市場價格，是否只用十進位輸出並避免保留原始非十進位格式？
